01 / 起因

"你今天已经生成了四十多张图了。每一张都差一点。
你也说不清楚差在哪里。"

这句话来自游戏里的一个事件节点。但它最初来自真实观察：


在AI工具普及之后，创意工作者的日常工作发生了一种微妙的结构性变化——不是"AI替代了人"，而是人开始用一种新的方式异化自己：不停生成、不停筛选、不停返修，却越来越难以解释"我为什么选了这一张"。


这个游戏想做的，不是批判AI，而是把这种体验可视化——把一个逐渐变得机械、重复、麻木的过程，压缩成几十分钟的玩法，让人在笑着点按钮的时候，隐约意识到：这好像也是我的日常。

观察 #1
"四十张图选一张"
判断力饱和，每张看起来都差不多
观察 #2
"忘记用了什么关键词"
过程与结果断开，无法复现
观察 #3
"说不清楚自己的思路"
被问到时发现无法回答
观察 #4
"反复粘贴同一组词"
不再思考，只是重复操作

02 / 理论框架

游戏的机制设计并非凭空而来，它建立在三个相互交叉的理论框架上。这些理论提供了一种语言，让"用AI工作感觉不对劲"这件事可以被更清晰地描述和分析。

理论 01
劳动异化
Entfremdung

马克思的异化理论描述了劳动者与其劳动产品、劳动过程、自身本质以及他人之间的四重分离。在AI创意劳动中，这种分离被加速和重新配置：工人不再是创造者，而是AI输出的筛选者。

Marx, 1844 经济学哲学手稿
理论 02
主体性异化
Alienation of Agency

当创造判断被持续外包给算法，个体的判断能力开始退化。Hannah Arendt区分了"劳动"（重复性）与"工作"（创造性）：AI工具正在将本属于"工作"的创意行为，转化为一种新形式的机械劳动。

Arendt, The Human Condition, 1958
理论 03
认知卸载
Cognitive Offloading

人类将认知任务外化到工具的行为本身是正常的——但当外化的是核心创意判断时，问题随之而来。Braverman的"技能剥夺"理论在这里获得了新的语境：不是体力，而是创造力本身在被逐步剥夺。

Braverman, Labor and Monopoly Capital, 1974

03 / 设计过程

从真实观察
到游戏机制

设计的核心工作是：把那些"说不清楚但确实存在"的感受，翻译成玩家可以感知的规则和选项。每一个游戏机制背后，都对应着一个真实的工作情境。

四十张图选一张 生成大量内容后，判断能力趋于饱和。所有图看起来都差不多好，又都差不多不好。
→
主体思考性下降

重复生成的选项会有括号注明"和上次一样"，主体思考性在每次无脑选择后扣减更多。

说不清楚这个方案的思路 项目通过后，被问到"为什么这么做"时，发现自己无法回答——那是AI生成的，当时觉得还行就用了。
→
阈值事件触发

思考性降到50时，触发"例会事件"：老板问创作思路，玩家只能选择"如实说"或"编一个"。

反复用同一提示词 改了七八次之后，不再想新思路，只是不停地粘贴上次的关键词，希望能出一张不同的。
→
按钮文字变化

前3次返修显示"重新思考提示词再生成"（-4思考），第4次起变成"同一提示词重新生成"（-9思考）。

随时想离职 在第7次被拒回来的时候，心里冒出来"不如不干了"，但同时又被整个系统的惯性裹着继续。
→
随机离职选项

第3次返修起，每次被老板拒绝有30%概率弹出"不想做了，直接离职"选项。

AI崩了反而松了口气 工具卡住的二十分钟，是少有的"自己做"的时间。但工具好了之后，那个状态又消失了。
→
工具报错事件

AI报错时出现选项：趁机自己起草稿（+8思考）vs 刷手机等（-7思考）。代表两种不同的反应。

04 / 核心数值设计

游戏只有两个可见数值——它们不是"生命值"，而是两种时间的压力：一种是内在的（思维还剩多少），一种是外在的（系统还能等多久）。

主体思考性
100 → 0

代表创作者还保有多少独立判断和主动思考的能力。不是体力，也不是情绪——是那个"我知道我在做什么、我为什么这么做"的感觉。

归零时：进入"空心人"结局。坐在屏幕前，输入框是空的，人也是空的。
延期率
0 → 100%

代表你在系统里还有多少时间和容错空间。每一次拒绝、每一次重来都在推高这个数字——它衡量的是"你在这个组织里还能撑多久"。

达到100%时：被系统淘汰。你的项目被转给另一个人，他用AI三天做完了。
通过率
5% → 28%

不是玩家的数值，而是老板"满意"的概率。基础5%，每次返修增加3%，封顶28%。这个设计本身就是一种立场——

用AI做的东西，通过率低，不是因为质量差，而是因为"对不对"从来都是模糊的。

05 / 三个结局

游戏没有"好结局"。三个结局代表了在AI工作压力下，人可能走向的三种不同方向——每一种都是真实的，每一种都有代价。

01
触发条件 — 主动选择离开
离职 / 寻找灵魂

你把工牌放在桌上，拎包走了。走到楼下，你站了一会儿。你还不知道下一步去哪里。但今天不用再打开AI工具了。

代表：清醒的抵抗。代价是真实的——失去了位置，但也许找回了主体性。这是唯一一个玩家可以主动触发的结局。
02
触发条件 — 主体思考性归零
空心人

你在输入框里输了几个字，然后停下来。你想不起来自己在做什么，也想不起来为什么要做。你不是累了。你只是空了。

代表：被动的崩溃。不是愤怒，不是抗拒——是那种更安静的、一点一点被磨空的感觉。对应Arendt意义上"工作"的彻底消失。
03
触发条件 — 任务最终通过
AI人

你已经看不出哪里是你的想法，哪里是AI给的。你不需要想从哪里开始。你打开工具，手指已经开始动了。这件事你做得很流畅。

代表：同化。这是最"成功"的结局，也是最沉默的异化。你赢了，但你赢的方式，让你成了另一种东西。

07 / 反思

"这件事你做得很流畅。"

游戏的最后一句话，放在"AI人"结局里——那是所有结局里唯一一个听起来像赞美的句子。流畅，高效，不出错，老板满意。

但它紧跟在一段让人不太舒服的描述后面：你已经看不出哪里是你的，你不需要想从哪里开始，你打开工具，手指已经开始动了。

这个游戏想问的问题不是"AI好不好"。它想问的是：当创造变得流畅，我们应该庆祝还是警惕？当我们再也不需要停下来想"我为什么这么做"，那个停下来想的时刻——是效率的损耗，还是人之为人的核心？

  • 如果你做的东西通过了，但你说不清楚为什么，那还算是你做的吗？
  • 熟练意味着进步，还是意味着某种东西的消失？
  • 你上一次有自己的想法是什么时候？
